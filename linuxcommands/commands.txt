Linux commands

 ctrl+alt+t - командаоткрыти терминала
 
 ~ - символ дом директории
 
 / - символ корневой директории
 
 . - обознасаеь текущую директорию
 .. - обозначает предыдущию директорию
 
1. ls - команда для вывода всех директорий (папка) и файлов в текущей (рабочей) директории втерминале 

2. cd (change dir) - команда для перехода из одной директории в  другую
 
3. mkdir - создает диреторию

4. touch - cоздает файл

5. rmdir - удал пустую директорию

6. rm - удал файл 

7. rm -rf - удал вообще все (ПРИНУД)

8.ls -a - показывает скрытые и обычные файлы директории
  ls -la - показывает подробную инфу
  
9.pwd(print working directory) - выводит путь до текущей директории

10.cp - копирует файл и директорию в указанное место
	cp <название> <куда>

11.mv - перемешает и переименовывание файла  или директории
		переим: mv oldname newname
		переме: mv name directory/
12.cat - показывает содержимое файла

13.nano - открывает файл для редактирования в графическом интерфейсе nano 
	*если нет файла, то нано создаст новый

14.echo - команда для быстрой записи текста в файл
	echo текст > файл
